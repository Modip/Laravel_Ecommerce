## Instalation de Laravel

Aller dans le site officiel de laravel ( https://laravel.com/docs/8.x/installation ). 
Aller à la partie installation via composer. Puis taper la commende 
"composer create-project laravel/laravel example-app
".
Pour la création d'un projet on tape la commende " laravel new nomDuProjet
"
Enfin on se position dans le dossier du projet et on fait la commande " php artisan serve
" pour lancé le projet.
